# EasyKernel
Easy and Fast Linux Kernel Compile Right on Your PC
# Requirements
• Installed gcc, git
• Linux kernel system
# Installation
```
git clone https://github.com/okinawa91/EasyKernel
cd EasyKernel
sudo chmod +x ./easykernel.sh
./easykernel.sh
```

P.S: it's not a real kernel compiler, just my first bash script:)
